Rails API for https://natural-db.com.

## Getting started

* Clone this repository:
```sh
$ git clone https://github.com/MangoSmoothie101010/natural-backend.git
````
* Change into the app directory:
```sh
$ cd natural-backend
```
* Install dependencies:
```sh
$ bundle install
```
* Setup database:
```sh
$ rails db:setup
```
* Start the application:
```sh
$ rails server
```
